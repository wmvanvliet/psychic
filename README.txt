Psychic is a Python package for EEG analysis and BCI construction.

It contains all the analysis code used in the doctoral thesis work of the authors.

Check some IPython notebooks that use psychic at: https://github.com/wmvanvliet/neuroscience_tutorials/tree/master/psychic
